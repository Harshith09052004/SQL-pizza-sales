# SQL Pizza Sales Analysis

## Project Overview
This project analyzes transactional pizza sales data using SQL to derive business insights.

## Objectives
- Calculate total revenue
- Identify top-selling pizzas
- Analyze revenue by category
- Track daily sales trends

## Tools Used
- MySQL
- SQL (JOIN, GROUP BY, ORDER BY, Aggregation)

## Key Concepts Demonstrated
- Aggregate functions
- Grouping and sorting
- Revenue computation
- Business data analysis

## Outcome
Demonstrates ability to extract meaningful business insights using structured SQL queries.
